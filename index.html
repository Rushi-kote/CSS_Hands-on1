What is CSS ?

  1. CSS stands for Cascading Style Sheet.
  2. CSS describe how HTML element are to be displayed on screen, paper, or in other media.
  3. Css is not neccesary CSS saves lot of work. It can control the layout of multiple web pages all at once
  4. External stylesheets are stored in CSS files.

CSS can be added to HTML documents in 3 ways:

  • Inline - by using the style attribute inside HTML elements
  • Internal - by using style element in the head section
  • External - by using a link element to link to an external CSS file.
  • The most common way to add CSS. os to keep the style in external CSS files. However, in this tutorial we will use inline and internal styles, beacouse this is easier to demonstrait, and easier for you to try it yourself.

When comparing CSS class vs ID, the difference is that CSS calss applies a style to multiple elements. ID, on the other hand, applies a style to one unique element. ID is aslo special in that you can use a special URL to link a derectly to an element and it's use by JavaScript

In CSS, selectors are used to target a specific element or range of elements on a web page. Once an element has been targeted, a style or set of style can be applied to the element.

There is a wide rage of selectors available. Two of the most commonly used are calss and ID. Both are used to target elements to which a style should be applied
